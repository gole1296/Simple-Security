# Simple Security

A comprehensive security management application for Microsoft Dataverse that simplifies user, team, and role administration. Manage security roles, field-level security profiles, and view detailed access permissions across your organization.

Built with React + TypeScript + Vite and preconfigured to work with Power Apps Code Apps.

## Quick Start (For Developers)

### Prerequisites
- Node.js 18+
- Power Platform CLI (pac)
- Authenticated to a Power Platform environment

### Development

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Deploy to Power Platform
pac code push
```

## Custom API

The application uses a custom Dataverse API `ope_simplesecurityaction` to perform security operations:

**Operations:**
- `associate` - Assign roles or profiles to users/teams
- `disassociate` - Remove roles or profiles from users/teams

**Parameters:**
- `ope_Operation` - Operation type (associate/disassociate)
- `ope_PrincipalType` - Principal type (systemuser/team)
- `ope_PrincipalId` - GUID of user or team
- `ope_RelatedType` - Related entity type (role/fieldsecurityprofile)
- `ope_RelatedId` - GUID of role or profile
